## Descripción

Material didáctico interactivo en formato web para la Unidad II del curso de Paradigmas de Programación. El contenido abarca la introducción a los lenguajes de programación modernos y multiparadigma, con énfasis en Python, JavaScript y PHP.

El material está diseñado como páginas HTML autocontenidas (sin dependencias de frameworks) que pueden abrirse directamente en cualquier navegador, sin necesidad de servidor.

---

## Contenido

### `index.html` — Introducción a los Lenguajes Multiparadigma

Presentación conceptual y teórica del tema central de la unidad:

| Clase | Tema |
|-------|------|
| 1 | ¿Qué es un Paradigma de Programación? |
| 2 | El Concepto Multiparadigma |
| 3 | Entornos CLI y Web |
| 4 | Python — Lenguaje Multiparadigma |
| 5 | JavaScript — Lenguaje Multiparadigma |
| 6 | PHP — Lenguaje Multiparadigma |
| 7 | Comparativa: Python, JavaScript y PHP |
| 8 | Beneficios del Multiparadigma |

### `javascript.html` — Introducción a JavaScript

Clases prácticas de JavaScript desde los fundamentos hasta el paradigma funcional:

| Clase | Tema |
|-------|------|
| 1 | Historia de JavaScript — Brendan Eich, ECMAScript, Node.js |
| 2 | Variables — var · let · const · Scope · Hoisting |
| 3 | Tipos de Datos — String · Number · Boolean · Null · Undefined · Array |

---

## Tecnologías Utilizadas

- **HTML5** — estructura semántica
- **CSS3** — diseño, variables CSS, animaciones, diseño responsivo
- **JavaScript (Vanilla)** — navegación entre secciones, interactividad del sidebar
- **Google Fonts** — EB Garamond, Inter, JetBrains Mono

No se utilizan frameworks externos ni librerías de terceros. Todo el material funciona offline.

---

## Cómo Usar

1. Clonar o descargar el repositorio
2. Abrir cualquier archivo `.html` directamente en el navegador (doble clic)
3. Navegar entre clases usando el menú lateral o los botones Anterior / Siguiente

```bash
git clone https://github.com/ingricardotoro/Paradigmas-Unidad-II-JavaScript.git
cd Paradigmas-Unidad-II-JavaScript
# Abrir index.html en el navegador
```

## Diseño

El diseño sigue un estilo editorial clásico con paleta cálida inspirada en papel pergamino:

- **Fondo:** tono crema / pergamino (`#f6efdd`)
- **Acento:** rojo oscuro académico (`#7a2418`)
- **Sidebar:** oscuro carbón (`#1a1610`)
- **Código:** tema oscuro con resaltado sintáctico personalizado
